# Interiors-App
React-Native App for wholesale interiors eCommerce.

⚡️ created using:
 * ReactNative
 * Expo
 * React-Navigation
 * Redux
 * Redux-Thunk
 * Firebase
 * MongoDB
 * ExpressJS
 * MongooseJS
 
 Full light ☀️ and dark 🌚 mode support!
 
 🔒 Full Firebase auth!
